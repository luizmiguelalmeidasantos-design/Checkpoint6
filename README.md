# 🎵 Sistema de Streaming de Música

## 📋 Funcionalidades
- Cadastro e gerenciamento de músicas e playlists
- Múltiplos tipos de usuários (Free e Premium)
- Sistema de reprodução completo
- Sistema de downloads para Premium

## 🏗️ Arquitetura
- Organização em pacotes (br.com.streaming.*)
- Conceitos de POO aplicados: Herança, Polimorfismo, Encapsulamento, Classes Abstratas e Interfaces.

## 🚀 Como Executar
Execute a classe `StreamingMusica.java` no pacote `br.com.streaming.principal`.
